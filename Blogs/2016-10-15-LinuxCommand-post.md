# 常用命令

* 路径移动: cd <path>
    * 移动到上一个文件夹: cd ..
    * 移动到根目录: cd ~
* 显示文件信息: ls
    * 显示完整的文件信息: ls -l
* 执行程序: ./<a.out>
* 打印内容: echo <some thing>
* 查看帮助文档: man <...>


# 编译程序

* 编译文件: gcc <file> 
    * 编译成指定文件: gcc <file> 

# 文件命令

* 在终端里输出文件内容: more <file>
